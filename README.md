# 🏆 비트코인 트레이딩 AI 전략 개발 프로젝트 결과 보고서

## 1. 프로젝트 개요
- **목표**: LSTM 모델을 활용하여 비트코인 가격 방향을 예측하고, 단순 보유(Buy & Hold) 대비 안정적인 수익을 창출하는 트레이딩 전략 수립.
- **기간**: 2023-08 ~ 2024-06 (테스트 데이터 기간)

---

## 2. 모델 설계 및 학습 성과
### 🔹 모델 아키텍처
- **구조**: 2-Layer LSTM (Hidden Size: 64)
- **규제 기법**: Dropout (0.5) 적용하여 과적합 방지
- **최적화**: Adam Optimizer (LR: 0.001), Early Stopping (Patience: 15)

### 🔹 학습 결과 분석
- **Loss 수렴**: Train Loss가 안정적으로 하락하였으며, Validation Loss와의 간격이 일정하게 유지되어 과적합이 효과적으로 제어됨.
- **Accuracy**: 훈련 정확도 약 **57%**, 검증 정확도 **50% 이상**을 달성하며 시장의 무작위성 대비 유의미한 예측력을 확보함.

---

## 3. 투자 전략 설계
### 🔹 핵심 전략
1. **적응형 임계값 (Threshold: 0.49)**: 데이터 분석을 통해 매수 신호가 드문 점을 고려, 최적의 진입 시점인 0.49를 임계값으로 설정.
2. **확률 기반 비중 조절 (Position Scaling)**: 모델이 예측한 상승 확률에 비례하여 투자 자금을 배분함으로써 하락 리스크를 관리함.
